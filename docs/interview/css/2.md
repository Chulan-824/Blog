# 2.flex: 1 是什么意思

flex: 1 为 flex: flex-grow flex-shrink flex-basis 简写

在 chrome 浏览器中flex: 1 `等价于` flex: 1 1 0%

- flex-grow 伸缩系数
- flex-shrink 收缩系数
- flex-basis 初始大小
