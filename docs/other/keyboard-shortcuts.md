# Keyboard Shortcuts

## Github Copilot

|              |             描述             |
| :----------: | :--------------------------: |
|     Tab      |           接受建议           |
|     Esc      |           拒绝建议           |
| Ctrl + Enter | 打开 Copilot Suggestion 面板 |
|  Option + ]  |          下一条建议          |
|  Option + [  |          上一条建议          |
| Option + \   |       触发行内 Copilot       |
